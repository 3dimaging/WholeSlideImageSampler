Module for sampling from a slide

Slide_Sampler

Slide_Sampler(self, wsi_file, desired_downsampling, size)

A WSI patch sampler.

Important are: self.wsi - an OpenSlide object of the multiresolution WSI specified by wsi_file. self.background_mask - a background mask (generate with self.add_background_mask()). Stored as a numpy array where 1.0 denotes tissue.

add_background_mask

Slide_Sampler.add_background_mask(self, desired_downsampling=32, threshold=4, disk_radius=10)

Add a background mask. That is a binary (0.0 vs 1.0), downsampled image where 1.0 denotes a tissue region. This is achieved by otsu thresholding on the saturation channel followed by morphological closing and opening to remove noise. The mask desired downsampling factor has a default of 32. For a WSI captured at 40X this corresponds to 1.25X. A moderate threshold is used to account for the fact that the desired downsampling may not be available. If an appropriate level is not found an exception is raised. :param desired_downsampling: :param threshold: :param disk_radius: for morphological opening :return:
